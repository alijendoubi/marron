# Marron. — Pâtisserie Artisanale · Tunis

> **Sweeten your day, the Marron way!**

Official website for **Marron.**, a Tunisian artisan pastry brand based in Tunis. The site serves as a digital storefront showcasing the full product catalogue and directing customers to order via WhatsApp.

🌐 **Live site:** [alijendoubi.github.io/marron](https://alijendoubi.github.io/marron)
📸 **Instagram:** [@marron.officialll](https://www.instagram.com/marron.officialll/)

## About Marron.

Marron. crafts premium artisan pastries in Tunis — from indulgent Layer Cakes and fudgy Brownies to crispy Croquant and wholesome Granola. Every creation is made daily with carefully sourced ingredients, balancing a warm artisanal feel with a premium finish.

## How Customers Order

1. Browse the catalogue on the website
2. Click **Commander** on any product category
3. A pre-filled WhatsApp message opens automatically
4. Confirm availability and delivery details with the team

No account, no cart, no checkout — frictionless ordering by design.

## Tech Stack

Built as a static site — no server, no build step, no dependencies to install.

- **HTML5 / CSS3 / Vanilla JS** — zero framework overhead
- **Three.js** — ambient particle animation in the hero
- **GSAP + ScrollTrigger** — scroll-driven entrance animations
- **Lucide Icons** — stroke icon set
- **Google Fonts** — Playfair Display, Lato, Josefin Sans
- **Hosted on GitHub Pages** — free, fast, zero maintenance
